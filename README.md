# Unexplored Telangana

A community-driven documentation project exploring the hidden culture, food, places, temples, art forms, tribes, and traditions of Telangana.

This repository aims to preserve and showcase:

- Unexplored tourist spots  
- Forgotten heritage sites  
- Lesser-known temples and deities  
- Rural and tribal cuisines  
- Indigenous art and craft forms  
- Cultural rituals and village traditions  

Contributions are welcome!
